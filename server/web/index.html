⚡ Quick Start

Connect from Python

Install the client and connect to this environment:

pip install openenv-core requests

Using OpenEnv client (via HTTP)

import requests

BASE = "http://localhost:8000"

# Reset the environment (start a new episode)
resp = requests.post(f"{BASE}/reset", json={"task": "priority-triage"})
obs = resp.json()["observation"]
print(obs["subject"])

# Take a step
action = {"priority": "urgent", "category": "technical", "reply_draft": ""}
result = requests.post(f"{BASE}/step", json=action).json()
print(f"Reward: {result['reward']}, Done: {result['done']}")

export HF_TOKEN="your-token-here"

📖 README

Customer Email Triage — OpenEnv

An OpenEnv environment where AI agents learn to triage customer support emails — a real task performed by thousands of support professionals daily.

🎯 Motivation

Customer support email triage is a high-value real-world task: companies receive hundreds to thousands of emails daily, requiring fast classification and routing. Misrouting or missed urgency costs time and money. This environment models that challenge faithfully.

🏆 Reward Function

Each step: reward = task_score − 0.02 (efficiency penalty)

Episode bonus: +0.10 if average step reward ≥ 0.80

  • priority-triage: score = priority_score (1.0 exact, 0.5 adjacent)
  • full-routing: score = 0.5×priority + 0.5×category
  • full-pipeline: score = 0.3×priority + 0.3×category + 0.4×reply_quality
